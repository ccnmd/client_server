# client_server
C-based client-server chat application for Ubuntu 22.04, supporting concurrent users, persistent chat logs, and basic data encryption.




# How to run:
  ## bash: 
       make
  
  ## Terminal 1:

    ##bash:
       ./server 

  ## Terminal 2:
 
    ##bash:
       ./client 
       #user name:  user2
       #password:   pass2
  ## Terminal ...
  ...


Note: All terninals are opened in the same folder of this project.
